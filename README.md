# GxTinyTerror.lv2
valve amplifier simulation


## Features

- Valve amplifier simulation. 
- Use it with a cabinet simulator after the amp.


## Dependencys

- libcairo2-dev
- libx11-dev

## Build
- git submodule init
- git submodule update
- make
- make install # will install into ~/.lv2 ... AND/OR....
- sudo make install # will install into /usr/lib/lv2

that's it.
